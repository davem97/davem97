First Person Character Controller

First Person Character Controller is a 3D Capsule based character controller. The aim of this
asset is to provide the following:

  • A character controller system that provides a wide range of features
  • A more straight-forward approach, accessible to everyone
  • A simple yet advanced base that handles the basics well
  • Default character controller concepts which have been tuned significantly
  • Different feel to a typical character controller
